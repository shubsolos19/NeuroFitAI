Technology

BUILT ON
REAL SCIENCE

NeuroFit uses Google's MediaPipe Pose — the same technology powering professional sports analytics — to detect 33 body landmarks with sub-pixel precision.

Height is calculated from head-to-ankle pixel span mapped to real-world scale using camera geometry and your setup parameters.

Weight estimation uses a clinically-validated height/BMI lookup table. Note: weight shown is a healthy reference value, not an actual measurement.

Meal recommendations come from a JS ML model ported from a Python Random Forest classifier, scoring 40 meals by calorie proximity, macro balance, and body-type cuisine preferences.
